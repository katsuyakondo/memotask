#47都道府県クイズ

## 紹介と使い方

  47都道府県の地図を基にどこの都道府県か4択の中から回答するクイズ。


## 工夫した点
localstorageを使って間違った回答を記憶させて次にその問題が登場した際に前回間違った問題である旨をアラート表示させた。
また、回答数を基にランキングを表示させた。


## 苦戦した点

  - 選択肢を漢字に変換するところがまだできていない。
  - また得点計算がうまくいっていないのでそこを修正したい。
